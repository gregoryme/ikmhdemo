IKMH Student Application Management System

The system is developed for the International Kids Media House (Vienna, Austria). The current version of the system is a demo version for testing and check the features.

 

Base conceptions

The entity USER is on the top of the hierarchy.

User's Email is used as a login and as a username in the table "users".

graph LR; USER-->ADMIN; USER-->APPLICANT; USER-->FACULTY_MEMBER;

 

Applicant is a person (senior school student, 14-17 y.o.) who applies for the IKMH educational program.

Faculty Member is a person who reviews and votes for the applications.

Admin is a person who controls the system and can retrieve some statistic information about the application process.

School Studio: as a rule, every applicants is a student (participant) of some school media studio.
